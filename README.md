# TMX Brand Kits

**Partner design-spec kits for the TMX ecosystem.**

</div>

---

Self-contained, reproduction-only design specs for partners running co-branded
campaigns with TMX ecosystem products. Each kit shows how to render a product's
brand so it reads as ours: the marks, colour, type, co-brand lockups, and the
usage terms. Every value is verified against the TMX design source of truth.

## Kits

## Reproduce, don't redraw

Use the supplied vector and raster files as-is. Never redraw, re-trace, recolour,
or AI-generate a mark or lockup. If a value ever disagrees with the live brand
source, the source wins.

> The kit pages render as designed once the repo is served over GitHub Pages.
> Until then, open `partner-kit.html` locally in a browser.
